표지 — FitFlow

"운동이 습관이 되는 순간, 삶이 달라진다 — AI와 함께라면 더 빠르게"
  • 프로젝트명: FitFlow — AI 맞춤 운동 습관 형성 서비스
  • 분류: AI 기반 헬스케어 웹 서비스
  • 타겟: 대학생 및 MZ세대 (18–34세)
  • 개발 형태: 1인 개발 / 3개월 이내 출시 목표
  • 핵심 기술: ChatGPT API · Bubble(노코드) · Google Sheets

프로젝트 개요

FitFlow는 사용자 데이터를 기반으로 AI가 매주 맞춤 운동 루틴을 생성하고, 챌린지와 피드백으로 지속 가능한 운동 습관을 만드는 웹 서비스입니다.

핵심 가치

개인 맞춤 · 지속성 · 접근성

기술 스택

ChatGPT API + Bubble + Airtable

수익 모델

Freemium (월 9,900원 프리미엄)

차별점

전문 지식 없어도 즉시 시작 가능

문제 정의

대학생의 72%는 '운동을 하고 싶지만 어떻게 시작해야 할지 모른다'고 응답합니다.
  • 운동 정보 과잉 — YouTube·SNS의 콘텐츠 홍수로 인해 오히려 실행 장벽 증가
  • 개인화 부재 — 체형·스케줄·목표가 각기 다른데 획일화된 루틴만 존재
  • 지속성 실패 — 동기 부여 없이 혼자 운동을 유지하기 어려움
  • 비용 문제 — 퍼스널 트레이닝 비용(월 30만~50만원)은 대학생에게 부담

타겟 사용자

페르소나 A — 운동 입문자

대학교 2학년 여성, 체중 관리가 목표. 헬스장 등록은 했지만 무엇을 해야 할지 몰라 포기한 경험 있음.

페르소나 B — 바쁜 직장인

25세 취업 준비생, 시간이 불규칙적. 집에서 할 수 있는 간단한 루틴이 필요함.

페르소나 C — 운동 중급자

대학원생 남성, 일정하게 운동하지만 프로그램화된 계획이 없어 성과가 정체됨.

  • 공통 키워드: 바쁜 일정 · 낮은 예산 · 스마트폰 친화 · 빠른 결과 원함

서비스 컨셉

"5분 입력 → AI 맞춤 루틴 생성 → 매일 챌린지 → AI 피드백" — 이 사이클이 습관을 만든다.
  • Input: 키·몸무게·목표·가용 시간·장비 여부 5가지 입력
  • Generate: ChatGPT API가 주 5일치 운동 루틴 + 식단 팁 생성
  • Challenge: 매일 완료 체크 & 스트릭(연속 달성) 시각화
  • Feedback: 주간 운동 데이터를 AI가 분석해 다음 주 루틴 자동 조정
  • Community: 같은 목표를 가진 사용자끼리 챌린지 공유

핵심 기능 (1인 개발 기준)

AI 루틴 생성기

사용자 정보 입력 시 ChatGPT API가 맞춤 운동 플랜 자동 생성

운동 체크리스트

일별 운동 완료 체크, 스트릭 카운터, 주간 달성률 시각화

AI 피드백 리포트

주 1회 운동 기록 분석 후 AI가 격려 메시지 + 다음 주 루틴 제안

운동 가이드 영상 링크

각 운동 동작별 YouTube 공식 영상 자동 연결

AI 기술 활용 방식

ChatGPT API(GPT-4o)를 핵심 엔진으로 활용 — 복잡한 ML 모델 없이도 개인화 달성
  • 루틴 생성: 사용자 정보를 System Prompt에 구조화 → GPT가 운동 플랜 JSON 반환
  • 피드백 생성: 주간 체크 데이터 → GPT가 격려 + 개선 포인트 자연어로 출력
  • Q&A 챗봇: 운동 관련 질문에 GPT가 실시간 답변
  • 식단 팁 생성: 운동 목표에 맞는 간단한 식단 제안 (영양사 수준 X, 참고용)
  • 비용 관리: 토큰 캐싱 + 루틴 재사용으로 API 비용 월 $20 이하 유지

  • 개발 비용 예상: Bubble 플랜 $29/월 + OpenAI API $20/월 = 약 7만원/월

콘텐츠 운영 전략

Project Overview

FitFlow란 무엇인가?

FitFlow는 사용자의 신체 정보와 생활 패턴을 5분 만에 입력하면 ChatGPT API가 주간 맞춤 운동 루틴을 즉시 생성해주는 AI 건강 웹 서비스입니다. 전문 트레이너 없이도 체계적이고 지속 가능한 운동 습관을 만들 수 있습니다.

Problem Definition

왜 지금 FitFlow인가?

대학생과 MZ세대가 운동을 포기하는 데는 명확한 구조적 이유가 있습니다.

01

정보 과잉으로 인한 결정 마비

유튜브·인스타의 운동 콘텐츠는 넘쳐나지만, 나에게 맞는 루틴을 고르는 것 자체가 어렵습니다. 결국 아무것도 시작하지 못합니다.

02

개인화된 운동 계획의 부재

체형·목표·가용 시간이 모두 다른데, 대부분의 서비스는 일반적인 루틴만 제공합니다.

03

높은 PT 비용

퍼스널 트레이닝 비용은 월 30~50만원 수준으로, 대학생에게 현실적이지 않습니다.

04

지속 동기 부여의 어려움

혼자 운동을 지속하는 것은 어렵습니다. 피드백이 없으면 3주 이내에 대부분 포기합니다.

Service Concept

서비스 컨셉

5분 입력 → AI 루틴 생성 → 매일 챌린지 → 주간 AI 피드백

이 단순한 사이클이 반복될수록 운동은 습관이 되고, 습관은 변화를 만듭니다.

📝

Step 1. 입력

신체 정보, 목표, 가용 시간, 장비 여부를 5분 내 입력

🤖

Step 2. AI 생성

ChatGPT API가 주간 맞춤 운동 루틴 즉시 생성

✅

Step 3. 실행

매일 운동 완료 체크 & 스트릭(연속 기록) 시각화

📊

Step 4. 피드백

AI가 주간 기록 분석 후 다음 주 루틴 자동 조정
