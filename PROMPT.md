# 건강문화 콘텐츠 기획안 리라이팅 프롬프트

기존 기획안(index.html / moodfit_plan.html)을 참고해 새로운 건강문화 콘텐츠 기획안을 작성할 때 사용하는 프롬프트입니다.

---

## 사용 방법

아래 프롬프트를 그대로 복사해 ChatGPT / Claude에 붙여넣고,
`[대괄호]` 안의 값만 원하는 내용으로 교체하면 됩니다.

---

## 마스터 프롬프트 — 기획안 리라이팅

```
다음 기획안을 참고해 비슷한 구조와 품질로 새로운 건강문화 콘텐츠 기획안을 작성해라.

---

참고 기획안 정보
- 기존 프로젝트 1: FitFlow (AI 맞춤 운동 루틴 생성 서비스)
- 기존 프로젝트 2: MoodFit (감정 × 운동 퍼플오션 전략 콘텐츠)
- 공통 구조: PPT 12슬라이드 + HTML 단일 페이지 + 인터랙티브 데모

---

새 기획안 조건

- 주제: [예: AI 수면 습관 개선 / 운동 커뮤니티 플랫폼 / 식단 × 감정 기록 등]
- 창의 전략: [블루오션 / 퍼플오션 / 문화 트렌드 결합 중 선택]
- 타겟: [대학생 / MZ세대 / 직장 초년생 등]
- 핵심 기술: [ChatGPT API / 노코드(Bubble) / SNS 챌린지 등]
- 개발 규모: 1인 개발, 3개월 이내 출시 가능 수준

---

기획안 구성 항목 (반드시 포함)

1. 프로젝트 제목 및 슬로건
2. 기획 배경 (문제 정의 + 데이터 근거)
3. 건강문화 트렌드 분석
4. 콘텐츠 컨셉 (한 문장 핵심 메시지 포함)
5. 콘텐츠 형식 (참여형 / 인터랙티브 / SNS 챌린지 등)
6. 타겟 사용자 (페르소나 2~3개)
7. 사용자 참여 방식 (단계별 흐름)
8. 콘텐츠 제작 방법 (기술 스택 + 월 운영 비용)
9. AI 활용 방식 (프롬프트 예시 포함)
10. 콘텐츠 운영 방식 (채널별 전략)
11. 콘텐츠 확산 전략 (해시태그 + 바이럴 루프)
12. 기대 효과 (정량 목표 포함)

---

출력 형식

1. 콘텐츠 기획 요약 (3~5줄)
2. 상세 기획안 (위 12개 항목)
3. PPT 발표자료 슬라이드 구조 (12슬라이드, 핵심 문장 중심)
4. HTML 단일 페이지 코드

HTML 요구사항:
- 기존 기획안과 동일한 레이아웃 구조 유지
  (nav / hero / 섹션별 카드 / PPT 슬라이드 뷰어 / footer)
- 카드형 UI + 반응형 레이아웃
- PPT 슬라이드는 키보드 방향키 전환 지원
- 인터랙티브 데모 섹션 포함 (클릭/선택 시 결과 표시)
- 기본 CSS 변수(:root) 기반 컬러 시스템 사용
- 단일 HTML 파일로 완성 (외부 라이브러리 사용 금지)

---

디자인 시스템 참고 (기존 기획안 공통 스타일)

색상 체계:
- Primary: 주제에 맞는 색상 1개 (예: 블루, 퍼플, 그린 등)
- Secondary: 보조 색상 1개
- Dark: #0F172A  /  Muted: #64748B  /  Light: #F8FAFC

컴포넌트:
- .card (hover 시 translateY(-4px))
- .tag (둥근 뱃지)
- .hero (그라디언트 배경 + 통계 표시)
- PPT 슬라이드 (.ppt.active 토글)
- 플로우 다이어그램 (단계별 화살표)

작성 스타일:
- 창의적이면서 현실적인 기획
- 대학생 프로젝트 수준 (운동처방학과 "건강문화콘텐츠기획론" 과제 기준)
- 이해하기 쉬운 설명, 발표용 핵심 문장 중심
```

## 빠른 변형 예시

위 프롬프트에서 `[주제]`만 바꿔 즉시 사용할 수 있는 예시입니다.

| 주제 | 창의 전략 | 핵심 기술 |
|------|---------|---------|
| AI 수면 습관 개선 서비스 | 블루오션 | ChatGPT API + 노코드 |
| 운동 × 음악 플레이리스트 | 퍼플오션 | Spotify API + AI |
| 식단 × 감정 기록 다이어리 | 퍼플오션 | ChatGPT + Airtable |
| 대학생 스트레칭 챌린지 SNS | 문화 트렌드 | 인스타 챌린지 + Canva |
| AI 명상 & 마음챙김 루틴 | 블루오션 | ChatGPT + Bubble.io |
| 팀 운동 챌린지 플랫폼 | 커뮤니티 전략 | Bubble + 소셜 기능 |
