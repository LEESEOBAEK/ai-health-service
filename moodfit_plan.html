기획 배경

왜 MoodFit인가?

현대 MZ세대는 신체와 감정을 함께 관리하는 통합 웰니스를 원하지만, 두 가지를 동시에 연결해주는 콘텐츠는 없었습니다.

68%

운동 포기의 가장 큰 이유

"운동할 기분이 아니다" — 대학생 10명 중 7명이 감정 상태로 인해 운동을 포기한 경험이 있습니다.

3배

무드 트래킹 앱 성장률

2022→2024년 감정 기록 앱 사용자 3배 증가. MZ세대의 자기감정 인식 욕구가 폭발적으로 성장 중입니다.

오운완

SNS 건강 인증 문화

"오늘 운동 완료" 인증 문화가 일상화. 운동 기록을 공유하고 서로 격려하는 건강 커뮤니티 문화 형성 중입니다.

공백

감정 × 운동 연결 콘텐츠 부재

기존 건강 콘텐츠는 신체 능력 중심. 감정 상태에 따른 맞춤 운동 콘텐츠는 아직 공백 시장입니다.

복잡한 ML 모델 없이, 프롬프트 엔지니어링만으로 고도화된 개인화 경험을 제공합니다.

프롬프트 예시 — 스트레스 상태 운동 루틴 생성

System: 당신은 운동처방사입니다. 사용자의 감정 상태에 최적화된 운동 루틴을 설계합니다.

User 정보:

출력 형식: JSON (운동명, 시간, 방법, 효과 포함)
